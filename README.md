# 게임창고 - 구글드라이브 연결형

기존 게임창고 디자인을 유지하면서, 카테고리 카드를 누르면 구글드라이브 폴더로 이동하는 버전입니다.

## 카테고리

1. 기타
2. 리소스팩
3. 맵
4. 모드
5. 모드팩
6. 서버

## 사용 방법

1. 압축을 풉니다.
2. `index.html` 파일을 브라우저로 엽니다.
3. 카테고리 카드를 누르면 연결된 구글드라이브 폴더가 새 창으로 열립니다.

## GitHub Pages에 올리는 방법

1. GitHub 저장소에 `index.html`, `style.css`, `script.js`, `README.md`를 업로드합니다.
2. Settings → Pages에서 배포 설정을 합니다.
3. 배포된 주소로 접속하면 됩니다.

## 링크 수정 방법

`script.js` 맨 위의 `categories` 배열에서 `url` 값을 바꾸면 됩니다.

## 중요

구글드라이브 폴더 공유 설정을 아래처럼 해야 방문자가 볼 수 있습니다.

- 일반 액세스: 링크가 있는 모든 사용자
- 권한: 뷰어
